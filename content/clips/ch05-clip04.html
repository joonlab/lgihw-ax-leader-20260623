🛠️ 이번엔 AI 기능이 들어간 웹앱을 직접 만듭니다 — 코드 한 줄 몰라도 됩니다. 무료 Gemini API 키를 발급받아, 회의록을 붙여넣으면 할일·담당·기한 표로 정리해주는 '회의록 정리기'를 Antigravity로 제작합니다. (앞 실습 ③이 백그라운드에서 도는 동안 진행합니다.)
① 무료 Gemini API 키 발급 → ② Antigravity에 복붙 프롬프트로 웹앱 제작 지시 → ③ 발급한 키를 넣고 테스트용 회의록으로 동작 확인. 외부 서버 없이 .html 파일 하나로 돌아가는 '내 전용 AI 도구'를 갖게 됩니다.
STEP 1 — 무료 Gemini API 키 발급 (약 2분)

웹앱이 AI를 호출하려면 본인의 Gemini API 키가 필요합니다. 신용카드·결제 등록 없이 무료로 받습니다.

1. 브라우저에서 aistudio.google.com/apikey 접속
2. 개인 Google(Gmail) 계정으로 로그인 (회사 계정이 막히면 개인 Gmail로)
3. 이용약관 동의 시 기본 프로젝트가 자동 생성됩니다 (결제 등록 불필요)
4. "API 키 만들기 (Get API key)" 클릭 → 생성된 키 옆 복사 아이콘(📋)으로 복사
5. 키는 보통 AIza...로 시작하는 긴 문자열입니다. 잠시 메모장 등에 붙여두세요.
📄 더 자세한 그림 설명은 다운로드 센터의 API키_발급가이드.md에 단계별로 정리돼 있습니다. 429 오류가 나면 무료 한도를 잠깐 넘은 것이니 1~2분 뒤 다시 시도하세요.
STEP 2 — Antigravity로 웹앱 제작

먼저 Gemini-API_레퍼런스.md 파일을 작업 폴더에 넣어두세요. (AI가 이 레퍼런스의 호출 패턴·규칙을 그대로 따라 만듭니다.) 그런 다음 아래 프롬프트를 붙여넣습니다.

📋 [복붙] 실습4_웹앱_01_회의록정리기제작.txt
(먼저 'Gemini-API_레퍼런스.md' 파일을 이 작업 폴더에 넣어두고 시작해줘.)

첨부한 'Gemini-API_레퍼런스.md'의 패턴과 규칙을 그대로 따라서,
'회의록 → 액션아이템 정리기' 단일 HTML 웹앱을 만들어줘.

요구사항:
- 화면 구성: ①Gemini API 키 입력란 ②회의록 붙여넣는 입력창(textarea) ③'액션아이템 추출' 버튼 ④결과 영역(요약 + 할일/담당/기한 표)
- 무료 Gemini API(gemini-3.5-flash)를 브라우저에서 직접 호출
- 회의록에 없는 내용은 지어내지 말고, 담당·기한이 없으면 '미정'으로 표시
- 외부 라이브러리 없이 .html 파일 하나로 동작하게
- 다 만들면 브라우저로 열어서 보여줘.
계획이 뜨면 확인하고 허용하세요. 잠시 후 단일 .html 웹앱이 만들어지고 브라우저로 자동으로 열립니다. 모델은 레퍼런스 기준 gemini-3.5-flash를 사용합니다 — 정리·추출에 균형이 좋고 무료 한도로 충분합니다.
🌐 만든 웹앱이 자동으로 안 열릴 때 — 'file URI'로 바로 열기
만든 .html이 브라우저로 자동으로 안 열릴 때가 있습니다. 그럴 땐 아래를 Antigravity 채팅창에 그대로 붙여넣어 file:// 주소(URI)를 받으세요. 받은 주소를 브라우저 주소창에 붙여넣으면 바로 열립니다.
📋 [복붙] HTML 바로 열기 (file URI)
방금 만든 HTML 파일을, 복사해서 브라우저 주소창에 붙여넣으면 바로 열리는 file:// URI(전체 경로)로 알려줘.
💡 우측 미리보기나 'Open in browser'가 안 뜰 때 특히 유용합니다. 받은 주소는 보통 file:///… 형태입니다.
STEP 3 — 키 넣고 테스트
1. 만들어진 웹앱의 ① API 키 입력란에 STEP 1에서 복사한 키를 붙여넣습니다.
2. ② 회의록 입력창에 아래 테스트용 회의록을 그대로 붙여넣습니다.
3. '액션아이템 추출' 버튼 클릭 → 요약 + 할일/담당/기한 표가 나오면 성공! 🎉
📋 [복붙] 실습4_웹앱_02_테스트용회의록.txt
[한울식품 영업팀 주간회의 - 2026년 6월 16일]
참석: 한상우 대표, 김민지(영업), 오태경(생산), 신동훈(물류)

1. GS리테일 PB 입점 건
- 김민지가 6월 20일까지 최종 견적서 제출하기로 함.
- 단가 협상은 대표님이 직접 진행.

2. 신메뉴 '들깨미역국 간편식'
- 생산 라인 점검 필요 → 오태경이 이번 주 내로 완료하기로.
- 초도 물량 5,000개를 6월 말까지 생산 목표.

3. 물류
- 행복유통 배송 지연 클레임 발생. 신동훈이 내일까지 원인 파악 후 보고.

4. 기타
- 7월 워크샵 장소 미정. 다음 회의 전까지 이수빈이 후보 3곳 조사하기로.
결과 표에서 담당·기한이 적힌 항목(예: 김민지 / 6월 20일)과, '미정'으로 처리된 항목(예: 7월 워크샵 장소)이 구분되는지 보세요. "없는 값을 지어내지 않고 미정으로 표시"하는 게 잘 동작하는 신호입니다.
✅ 막힐 때 체크포인트 & 무료 한도(쿼터)
결과가 안 나오면 순서대로 확인하세요 — 대부분 여기서 풀립니다.
  • 키 형식: 입력한 키가 AIza…로 시작하고, 앞뒤 공백 없이 붙었는지.
  • 모델명: 코드의 모델이 gemini-3.5-flash인지 (오타·구버전명이면 호출 실패).
  • 무료 한도(429): 무료 키는 분당 요청 수에 제한이 있어, 여러 명이 동시에 누르면 429(한도 초과)가 납니다. 1~2분 뒤 다시 누르거나, 옆 사람과 시차를 두고 실행하세요.
  • 그래도 안 되면: 브라우저 F12 → Console 탭의 빨간 오류 메시지를 그대로 Antigravity에 붙여넣고 "고쳐줘"라고 하면 됩니다.
⚠️ API 키는 비밀번호입니다 — 절대 공유 금지
발급한 키는 비밀번호와 동급의 개인정보입니다. 채팅·메일·캡처로 타인에게 공유하지 마세요. 화면 공유 시 키 입력란이 노출될 수 있으니 주의하고, GitHub 등 공개 코드에 절대 올리지 마세요. 이 웹앱은 단일 HTML이라 입력한 키가 본인 브라우저 안에서만 쓰이고 어떤 서버에도 저장·전송되지 않습니다. 보안이 걱정되면 실습 후 AI Studio에서 해당 키를 삭제해도 됩니다.
