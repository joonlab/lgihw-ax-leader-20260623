Gemini 주요 기능
  • Gemini 채팅: 자연어로 AI와 대화하며 업무를 처리하는 기본 인터페이스
  • Deep Research: 심층 조사 모드 — AI가 자율적으로 여러 소스를 탐색하고 종합 보고서를 작성
  • Gems: 맞춤형 AI 에이전트 — 특정 역할과 지침을 설정하여 재사용 가능한 AI 전문가 생성
  • Google 연동: Gmail, Drive, Docs, Sheets 등 구글 워크스페이스와 직접 연결
  • 멀티모달: 텍스트, 이미지, PDF, 동영상, 코드 저장소 등 다양한 입력을 동시 처리
  • Thinking Level: 추론 깊이를 조절 (low / medium / high) — 속도와 품질 균형 조절 가능
접속 방법
  • 웹: gemini.google.com 접속
  • Google 계정 로그인 후 바로 사용 시작
  • 좌측 상단 모델 선택 드롭다운에서 3.1 Pro / 3.5 Flash / 3.1 Flash-Lite와 사고 수준(Thinking) 선택

Gemini 홈 화면 — 하단 입력란에 질문을 입력하고, 왼쪽 + 버튼으로 파일을 첨부합니다 (실측 캡처 2026.06.23)

모델 선택 — 3.1 Flash-Lite(신규), 3.5 Flash, 3.1 Pro 및 사고 수준(Standard)을 선택합니다

Gems 관리 — Google 사전 제작 Gem(Storybook, 브레인스토밍, 커리어 컨설턴트, 코딩 파트너)과 내가 만든 Gem을 관리합니다

좌측 메뉴 — 새 채팅, 이미지, 라이브러리, Gems, 노트북(NotebookLM 연동)에 접근합니다

Gemini 홈 화면 — 프롬프트 입력란에 질문을 입력하고, 왼쪽 하단의 + 버튼으로 파일을 첨부할 수 있습니다

모델 선택 — 우측 드롭다운에서 Gemini 3, 빠른 모드, 사고 모드, Pro 중 선택합니다

파일 첨부 메뉴 — + 버튼을 누르면 파일 업로드, Drive 파일 추가, 코드 가져오기 옵션이 나타납니다

구독 플랜 안내
무료: Gemini 3.5 Flash 기본 사용. AI Pro ($20/월): Gemini 3.1 Pro, Deep Research, 확장 컨텍스트. AI Ultra ($250/월): 최고 한도 + NotebookLM Plus + 최신 기능 우선 접근. 오늘 실습의 기본 모델은 3.5 Flash입니다 — 빠르고 안정적이라 대부분의 실습에 충분합니다. 고난도 추론이 필요할 때만 Pro를 선택하고, Pro에서 오류가 잦으면 즉시 3.5 Flash로 내려서 진행하세요.
