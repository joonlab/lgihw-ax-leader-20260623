🔎 이 실습은 웹 자료조사를 AI에게 통째로 맡기는 체험입니다. 단, 딥리서치는 시간이 오래 걸리므로 먼저 실행만 걸어두고 곧바로 실습 ④를 병렬로 진행합니다. ④를 어느 정도 하고 돌아오면, 출처 링크가 달린 HTML 보고서가 완성돼 있습니다.
"생성형 AI 업무활용 트렌드 2026"을 주제로, 신뢰할 수 있는 출처(McKinsey·Gartner·Deloitte 등)를 AI가 직접 웹에서 찾아 정리 → 출처 링크가 클릭되는 HTML 보고서 한 파일로 받습니다. 시간이 걸리는 약점을 '병렬'로 역이용하는 게 이 실습의 묘미입니다.
⏱️ 진행 순서 — 이 실습만 흐름이 다릅니다

다른 실습은 "넣고 → 기다렸다 결과 확인"이지만, ③은 실행을 걸어둔 채 다른 실습을 하고 나중에 돌아오는 구조입니다. 오늘 시간표에도 두 칸으로 나뉘어 있습니다.

시각할 일
14:55 (착수)아래 복붙 프롬프트를 넣어 자료조사를 '시작'만 시켜둔다 → 바로 실습 ④로 이동
15:05~15:35④ 웹앱을 만드는 동안 ③은 백그라운드에서 알아서 웹을 뒤지며 보고서를 작성 중
15:35 (확인)③으로 돌아와 완성된 HTML 보고서를 확인 · 출처 링크 클릭해보기
한 사람이 두 가지 일을 동시에 굴리는 경험입니다. 사람은 ④에 집중하고, 시간이 오래 걸리는 조사는 AI에게 맡겨두는 것 — 실제 업무에서 가장 체감이 큰 활용법입니다.
STEP 1 — 자료조사 '실행만' 걸어두기

Antigravity 입력창에 아래 프롬프트를 그대로 붙여넣고 실행합니다. 계획이 뜨면 확인 후 진행을 허용하세요. 그러면 AI가 웹 검색을 시작합니다 — 여기서 기다리지 말고 곧바로 실습 ④로 넘어갑니다.

📋 [복붙] 실습3_리서치_01_트렌드조사HTML.txt
'생성형 AI 업무활용 트렌드 2026'을 주제로 웹에서 자료를 조사해줘.

조건:
- McKinsey, Gartner, Deloitte 같은 신뢰할 수 있는 출처를 찾아줘.
- 핵심 트렌드를 7~8개 정리하고, 각 트렌드마다 출처(기관명 + 링크)를 반드시 포함해줘.
- 출처가 없는 내용은 넣지 마.

조사가 끝나면 그 내용을 깔끔하고 보기 좋은 HTML 보고서 한 파일로 만들어줘:
- 제목, 핵심 요약, 트렌드별 섹션, 각 섹션에 출처 링크(클릭 가능)
- 다 만들면 브라우저로 열어서 보여줘.

(참고: 이 작업은 자료 조사에 시간이 좀 걸려. 먼저 시작시켜두고, 돌아가는 동안 다음 실습을 진행하면 돼.)
💡 AI가 웹을 직접 뒤지게 하려면 웹 검색 기능이 켜져 있어야 합니다. Antigravity에서는 /browser 명령이나 웹검색 도구를 활용하면 더 정확한 출처를 가져옵니다. 실행이 시작됐는지(검색·페이지 열람 로그가 도는지)만 확인하고 넘어가세요.
STEP 2 — (15:35) 돌아와서 결과 확인
1. ④를 진행하다 ③ 탭/대화로 돌아오면, 보고서 HTML이 자동으로 열려 있거나 작업 폴더에 .html 파일이 생성돼 있습니다.
2. 트렌드 7~8개가 섹션으로 정리됐는지, 각 섹션에 출처 링크가 달렸는지 확인합니다.
3. 출처 링크를 실제로 클릭해 원문(보고서·기사)이 열리는지 확인하세요. — "근거가 살아있는" 보고서인지 검증하는 단계입니다.
완성 수준이 궁금하면 먼저 비교해 보세요. 실습자료 다운로드 센터의 "샘플 HTML 보고서"가 미리 만들어둔 완성본입니다. "이런 수준이 나옵니다"의 기준점으로 삼으면 됩니다.
✅ 완성 기준
트렌드 7~8개 + 각 항목마다 클릭 가능한 출처 링크가 달려 있다.
브라우저로 바로 열리는 깔끔한 HTML 한 파일이다. (제목·핵심 요약·트렌드별 섹션 구성)
출처가 없는 내용은 들어가지 않았다. (근거 없는 문장 = 환각, 넣지 않기)
⚠️ 강사 팁 — 시간을 '무기'로 바꾸세요
  • 딥리서치는 느린 게 약점이지만, 이 실습에선 그 시간을 ④와 병렬로 굴려 오히려 강점으로 바꿉니다. "사람 1명이 일 2개를 동시에 돌린다"를 체감시키세요.
  • 착수 직후 완성 샘플을 먼저 보여주고 "이 정도가 나옵니다"로 기대치를 잡으면, ④ 하는 동안 불안해하지 않습니다.
  • 출처 링크가 실제로 열리는지 한두 개 같이 클릭해보세요 — "AI가 지어낸 게 아니라 진짜 근거"라는 신뢰가 핵심 메시지입니다.
