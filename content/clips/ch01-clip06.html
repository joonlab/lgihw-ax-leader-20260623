본격적인 실습에 들어가기 전, "AI 에이전트가 실제로 일하는 모습"을 강사 시연으로 먼저 봅니다. 여기서 보는 도구는 Claude Code — 터미널에서 동작하는 AI 에이전트입니다. 오늘 여러분이 직접 따라할 필요는 없습니다. "AI에게 이런 것까지 맡길 수 있구나"를 눈으로 확인하는 시간입니다.
이 페이지는 "시연 관전용"입니다
설치나 명령어를 외우지 마세요. 아래 3가지 시연은 약 15~20분 동안 강사가 화면을 공유하며 보여줍니다. 여러분은 ① AI가 무엇을 스스로 판단하는지 ② 어디서 사람의 확인을 받는지만 관찰하면 됩니다. 오후에는 비슷한 일을 Antigravity로 직접 해보게 됩니다.
Claude Code가 일반 AI 채팅과 다른 점

Gemini·ChatGPT 같은 채팅은 답변(텍스트)을 돌려줍니다. Claude Code는 거기서 한 발 더 나아가, 내 컴퓨터에서 실제로 작업을 수행합니다.

구분일반 AI 채팅Claude Code (에이전트)
결과물화면에 글로 답변실제 파일 생성·수정·정리
도구 사용없음 (대화만)웹 검색, 파일 열기, 코드 실행 등 자율 사용
작업 범위한 번의 질문·답변목표를 주면 여러 단계를 스스로 반복
사람의 역할매번 다음 질문 입력목표 제시 + 중요한 순간에 승인
핵심 한 줄
채팅은 "말해주는" AI, 에이전트는 "대신 해주는" AI입니다. 아래 3가지 시연은 모두 "대신 해주는" 장면입니다.
시연 1 웹 크롤링 → 데이터 정리
약 5분 데이터·전략 / 마케팅 직무 공감대

웹페이지에 흩어진 정보를 사람이 일일이 복사·붙여넣기 하던 일을, AI가 방문 → 추출 → 표로 정리 → 파일 저장까지 한 번에 처리합니다.

강사가 입력하는 명령 (예시)
아래 페이지들을 방문해서 제품명·가격·출시일을 추출하고, 하나의 표로 정리해서 비교.csv 파일로 저장해줘. - (URL 1) - (URL 2) - (URL 3)
AI가 스스로 하는 일
  • 각 페이지를 순서대로 방문해 필요한 항목만 골라 읽기
  • 형식이 제각각인 데이터를 같은 기준으로 맞추기
  • 표(CSV)로 정리해 파일로 저장 → 엑셀에서 바로 열람 가능
관전 포인트
사람은 "무엇을 모을지"만 말했을 뿐, "어떻게 모을지"는 AI가 스스로 정합니다. 페이지 구조가 달라도 알아서 대응하는 장면을 눈여겨보세요.
시연 2 로컬 파일 일괄 정리 / 이름 변경
약 5분 전 직무 공통 — "AI가 내 PC를 정리한다"

다운로드 폴더에 뒤섞인 수십 개의 파일을 종류별로 분류하고, 규칙에 맞게 이름을 바꾸는 지루한 작업을 AI에게 맡깁니다.

강사가 입력하는 명령 (예시)
이 폴더의 파일들을 확장자별 폴더로 나누고, 사진 파일은 "촬영일_순번" 형식으로 이름을 바꿔줘. 바꾸기 전에 어떻게 바꿀지 먼저 목록으로 보여줘.
AI가 스스로 하는 일
  • 폴더 안 파일을 훑어 종류(문서·이미지·압축 등) 파악
  • 새 폴더를 만들고 파일을 옮길 계획을 먼저 제시
  • 사람이 "진행"이라고 확인하면 그때 실제 변경 실행
관전 포인트
파일을 실제로 바꾸기 직전에 "이렇게 바꿔도 될까요?"라고 사람에게 확인을 받습니다. 에이전트가 위험한 작업 앞에서 멈추고 승인을 구하는 — 안전장치(Human-in-the-loop)를 꼭 확인하세요.
시연 3 종합 자료 기반 보고서 초안 작성
약 7분 상품기획 / 영업 / CXM — 문서 작업 즉시 활용

여러 개의 자료(문서·표·메모)를 한꺼번에 읽고, 구조를 갖춘 보고서 초안으로 엮어냅니다. 앞의 시연 1·2 결과물까지 재료로 쓸 수 있습니다.

강사가 입력하는 명령 (예시)
이 폴더의 자료들을 모두 읽고, 팀 공유용 1페이지 보고서를 markdown으로 작성해줘. 구성: 핵심 요약 → 주요 발견 3가지 → 다음 액션. 표와 불릿을 적절히 섞어줘.
AI가 스스로 하는 일
  • 여러 파일을 모두 열어 내용을 종합적으로 이해
  • 요청한 보고서 구조(요약·발견·액션)에 맞춰 재구성
  • 완성된 초안을 파일로 저장 → 바로 다듬어 공유 가능
관전 포인트
단순 요약이 아니라 "내가 정한 형식(요약→발견→액션)"에 맞춰 정리한다는 점이 핵심입니다. 앞에서 배운 프롬프트 4원칙(역할·맥락·형식·단계)이 그대로 적용되는 장면을 찾아보세요.

이 섹션의 핵심
AI 에이전트(Claude Code)는 웹 크롤링·파일 정리·보고서 작성을 사람 대신 스스로 수행합니다. 채팅이 "말해주는" AI라면, 에이전트는 "대신 해주는" AI입니다. 위험한 작업 앞에서는 사람의 승인을 받는다는 점(안전장치)을 기억하고, 오후 Antigravity 실습으로 이 감각을 직접 체험하세요.
