🗂️ 오늘의 첫 실습이자 워밍업입니다. "Antigravity가 내 PC의 파일을 직접 만지고 정리한다"를 몸으로 체감하는 단계예요. 가상회사 한울식품(주)의 뒤섞인 업무 파일 27개를 — 종류별로 자동 분류 → 파일명 규칙 통일 → 중복 정리 — 세 단계로 깔끔하게 정돈합니다.
더미 데이터 폴더를 작업 폴더로 연 뒤, 채팅에 복붙 프롬프트 3개를 순서대로 붙여넣습니다. AI가 "이렇게 옮기겠다"는 계획을 표로 먼저 보여주면 확인하고 → "진행해"로 실제 실행. "계획 확인 → 허용 → 실행"의 안전 루틴을 처음 익히는 실습입니다.
준비 — 더미 데이터 폴더 열기
1. 자료실에서 더미 데이터 zip(한울식품_업무자료.zip)을 받아 압축을 풉니다.
2. Antigravity에서 압축 푼 폴더를 작업 폴더(프로젝트)로 엽니다. 이 폴더가 AI의 작업 무대가 됩니다.
3. 매출·지출·회의록·보고서 파일이 종류 구분 없이 한곳에 뒤섞여 있는 상태를 눈으로 확인합니다. (정리 전 모습)
STEP 1 — 종류별 자동 분류

먼저 어질러진 파일들을 종류별 폴더로 나눕니다. 아래 프롬프트를 채팅창에 그대로 붙여넣으세요.

📋 [복붙] 종류별 폴더로 자동 분류
내가 방금 '한울식품(주) 업무자료' 폴더의 파일들을 여기 작업 폴더에 풀어놨어.
지금 매출, 지출, 회의록, 보고서, 공지 등 여러 종류의 파일이 구분 없이 한곳에 뒤섞여 있어.

아래 종류별 폴더를 만들어서 각 파일을 알맞게 자동으로 분류해줘:
01_매출 / 02_지출회계 / 03_거래처 / 04_회의록 / 05_보고서 / 06_인사 / 07_재고물류 / 08_공지일정 / 09_고객 / 10_기타

규칙:
- 파일을 실제로 옮기기 전에, 어떤 파일을 어느 폴더로 보낼지 먼저 표로 정리해서 보여줘.
- 내가 "진행해"라고 하면 그때 실제로 이동해줘.
- 어느 폴더에 넣을지 애매한 파일은 따로 표시해서 알려줘.
이렇게 흘러갑니다: AI가 "어느 파일을 어느 폴더로 보낼지" 표(계획)를 먼저 보여줍니다. 표를 확인하고 이상이 없으면 채팅에 "진행해"라고 답하세요. 그제서야 실제로 폴더가 만들어지고 파일이 이동합니다. (실행 직전 [허용/실행] 버튼이 뜨면 눌러 주세요.)
💡 이렇게 응용해 보세요 — 폴더 이름만 내 업무에 맞게 바꾸면(예: 계약서 / 견적 / 정산) 내 다운로드 폴더·바탕화면 정리에 그대로 씁니다.
STEP 2 — 파일명 규칙 통일

분류가 끝났으면, 제각각인 파일 이름(0603, 6월, 2026-05-12가 섞임)을 하나의 규칙으로 통일합니다.

📋 [복붙] 파일명을 한 가지 규칙으로 통일
분류한 파일들의 이름 형식이 제각각이야. (날짜가 0603, 6월, 2026-05-12 처럼 섞여 있고, 한글 이름과 영어 이름도 혼재해.)

파일 이름을 아래 규칙으로 통일해서 바꿔줘:
- 날짜는 모두 YYYY-MM-DD 형식으로
- '[종류]_YYYY-MM-DD_내용' 형태로 (예: 매출_2026-05-31_5월매출정리)
- 날짜 정보가 없는 파일은 날짜 없이 '[종류]_내용'으로

먼저 '기존 이름 → 새 이름' 변경안을 표로 보여주고, 내가 "진행해"라고 하면 실제로 이름을 바꿔줘.
💡 이렇게 응용해 보세요 — 우리 팀이 쓰는 파일명 규칙(예: [프로젝트]_[버전]_[날짜])을 그대로 적어주면, 협업 폴더의 이름 정돈에 바로 활용할 수 있습니다.
STEP 3 — 중복 파일 정리

마지막으로 복사본_처럼 내용이 같거나 비슷한 중복 파일을 찾아 정리합니다.

📋 [복붙] 중복 파일 찾아 정리 추천
이 폴더 안에 내용이 같거나 거의 비슷한 중복 파일이 있는지 찾아줘.
(예: '복사본_'으로 시작하는 파일)

찾으면:
- 어떤 파일들이 서로 중복인지 묶어서 보여주고
- 무엇을 남기고 무엇을 지우는 게 좋을지 추천해줘.

실제 삭제는 내가 확인한 다음에 해줘.
💡 이렇게 응용해 보세요 — "삭제는 내가 확인한 다음에"라는 한 줄이 안전장치입니다. 실수로 지우면 되돌리기 어려우니, 중요한 폴더일수록 이 한 줄을 꼭 붙이세요.
⚠️ 따라하기 핵심 — "옮기기 전에 계획부터"
  • AI는 파일을 실제로 옮기기 전에 계획(표)을 먼저 보여줍니다. 한 번 검토하고 "진행해"로 승인하는 흐름을 몸에 익히세요.
  • 실행 직전 [허용/실행] 버튼이 뜨면 내용을 확인하고 눌러야 실제 작업이 일어납니다. 묻지도 않고 다 해버리는 도구가 아니라, 내가 통제권을 쥐고 있다는 점이 포인트.
  • 한 번에 너무 많이 시키지 말고 STEP 1 → 2 → 3 순서대로 단계적으로 진행하세요.
