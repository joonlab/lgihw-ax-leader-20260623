비즈니스 프롬프트 연습

아래 프롬프트들을 Gemini에 직접 입력해 보세요. 이런 블록 부분을 본인의 상황에 맞게 수정하면 더 정확한 결과를 얻을 수 있습니다.

보고서·자료 분석 및 요약
당신은 첨부 자료를 정확하게 읽고 핵심을 뽑아내는 분석 전문가입니다. 첨부한 보고서 / 회의록 / 기사 등을 분석해 주세요. 다음 형식으로 정리해 주세요: 1. 핵심 요약 (3줄 이내) — 결론을 먼저 제시 2. 현황 진단 (표 형태) — 항목 / 현재 상황 / 시사점 3. 주요 쟁점 3가지 — 각 쟁점의 배경과 영향 4. 실행 가능한 액션 3가지 — "누가, 언제까지, 무엇을" 5. 추가 확인이 필요한 사항 — 근거가 부족하거나 검증이 필요한 부분
회의 준비 브리핑
당신은 회의 준비를 돕는 기획 담당자입니다. 회의 유형: 팀 회의 / 부서 회의 / 파트너 미팅 등 회의 주제: 회의 주제 입력 참석자: 주요 참석자와 역할 회의 시간: 예: 60분 다음 내용을 포함한 브리핑 자료를 작성해 주세요: 1. 회의 배경 및 목적 (3줄) 2. 핵심 안건별 요약 — 현황, 쟁점, 권장 방향 3. 참석자별 예상 입장과 반대 의견 대응 논리 4. 의사결정 포인트 (예/아니오) 5. 발언 제안 — 오프닝 멘트 + 질문 3~5개 6. Follow-up 액션 템플릿
마케팅 카피·홍보 문구 작성
당신은 전환율 높은 카피를 쓰는 마케팅 카피라이터입니다. 홍보할 대상: 제품/서비스/캠페인 입력 타겟 고객: 예: 30대 직장인 / 신규 가입자 등 채널: 예: 문자(SMS) / SNS / 이메일 / 배너 핵심 메시지: 전달하고 싶은 1가지 다음을 작성해 주세요: 1. 헤드라인 후보 5개 (서로 다른 톤으로) 2. 본문 카피 — 짧은 버전(SNS용) + 긴 버전(이메일용) 3. CTA(행동 유도) 문구 3개 4. 각 후보별로 어떤 고객 심리를 자극하는지 한 줄 설명
고객 응대·이메일 답장 초안
당신은 정중하고 명확하게 소통하는 고객 응대 담당자입니다. 아래 고객 문의(또는 메일)에 대한 답장 초안을 작성해 주세요. 받은 내용: 고객 문의 / 메일 원문 붙여넣기 상황: 예: 불만 접수 / 단순 문의 / 환불 요청 등 원하는 톤: 정중한 격식체 / 친근한 안내체 다음 조건으로 작성해 주세요: 1. 공감 → 사실 확인 → 해결 방안 → 마무리 순서 2. 과도한 약속이나 확정되지 않은 정보는 넣지 말 것 3. 핵심은 3문장 이내로 간결하게 4. 톤이 다른 2가지 버전 제시 (간결형 / 상세형)
참고
이 프롬프트 기법은 Gemini에서만 사용할 수 있는 것이 아닙니다. ChatGPT, EXAONE, Claude 등 모든 AI에서 동일하게 적용됩니다. 오늘은 Gemini로 실습하지만, 익힌 원칙은 어떤 AI에서든 활용하세요.
Gemini Canvas로 문서 작업 단축하기

Gemini의 캔버스(Canvas)는 대화창과 분리된 편집 공간입니다. AI가 생성한 리포트, 인포그래픽, 웹페이지를 캔버스 위에서 직접 수정·보완하고, 원하는 부분만 선택하여 추가 지시를 내릴 수 있습니다. 문서 작업의 혁신적 단축을 체험해 보세요.

캔버스 사용법
AI 답변 하단의 "캔버스에서 열기" 버튼을 클릭하면 편집 모드로 전환됩니다. 텍스트를 직접 수정하거나, 특정 부분을 드래그하여 "더 간결하게", "표로 변환" 등의 추가 지시를 내려 보세요.
실습 중 자주 막히는 지점
· 부분 수정이 엉뚱하게 적용될 때 — 예를 들어 "1번 항목만 영어로"라고 했는데 전체가 영어로 바뀌는 경우가 있습니다. 어느 부분인지 더 구체적으로 짚어서("위에서 두 번째 문단만") 다시 요청하면 정확히 동작합니다.
· 슬라이드·PPT 변환이 자꾸 실패할 때 — Gemini 서버가 붐비면 변환이 멈출 수 있습니다. "다시 시도" 버튼을 누르거나, 안 되면 웹페이지·인포그래픽 형식으로 우회하세요.
· 결과가 점점 이상해질 때 — 맥락이 오염된 신호입니다. 같은 대화를 붙잡고 씨름하기보다 새 채팅에서 다듬은 프롬프트로 다시 시작하는 것이 빠릅니다.
