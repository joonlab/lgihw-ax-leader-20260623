AI에게 구조화된 프롬프트를 작성할 때 자주 사용하는 마크다운(Markdown) 문법을 총정리합니다. 한 페이지에서 필요한 문법을 모두 확인하세요.
마크다운이란?

마크다운(Markdown)은 간단한 기호로 문서 구조를 표현하는 방법입니다. 2004년 존 그루버(John Gruber)가 만든 이 문법은 현재 ChatGPT, Claude, Notion, Obsidian, GitHub 등 거의 모든 AI·협업 도구에서 표준으로 사용됩니다.

마크다운을 사용하면 AI가 프롬프트의 구조를 더 명확하게 이해하여 더 정확하고 체계적인 결과를 만들어냅니다.

이 페이지 활용법
각 섹션의 난이도 표시를 참고하세요: 필수 꼭 알아야 할 핵심 문법   실용 업무에서 유용한 문법   참고 필요할 때 찾아보는 문법
제목 (Headings) 필수

# 기호 뒤에 공백 한 칸을 두고 텍스트를 쓰면 제목이 됩니다. #의 개수가 많을수록 작은 제목입니다.

입력 방법렌더링 결과업무 활용
# 제목 제목 보고서 제목, 프롬프트의 대주제
## 소제목 소제목 챕터, 섹션 구분
### 세부 제목 세부 제목 소주제, 세부 항목
#### 하위 제목 하위 제목 세부 분류 항목
##### 소항목 소항목 드물게 사용
###### 최소 제목 최소 제목 거의 사용하지 않음
핵심 구분: 제목 vs 태그
# 제목  (# 뒤에 공백) → 제목으로 인식됩니다.
#남산맛집  (# 뒤에 공백 없음) → SNS의 태그(해시태그)로 인식됩니다.
AI 프롬프트에서 제목을 만들 때는 반드시 # 뒤에 공백을 넣으세요.
업무 문서 계층 매핑
# 보고서 제목 → ## 챕터 → ### 소제목 → #### 세부항목
실무에서는 # ~ ### 3단계면 충분합니다.
서식 (Formatting) 필수

텍스트를 굵게, 기울여, 취소선으로 강조할 수 있습니다.

서식입력 방법렌더링 결과용도
굵게 (Bold) **중요한 내용** 중요한 내용 핵심 단어 강조
기울임 (Italic) *참고 사항* 참고 사항 부연 설명, 용어 표시
취소선 ~~이전 내용~~ 이전 내용 수정 전후 비교
굵게+기울임 ***매우 중요*** 매우 중요 최고 수준 강조
입력
이번 분기 실적은 **전년 대비 23% 성장**했으며,
특히 *동남아 시장*에서 큰 성과를 보였습니다.
~~기존 목표 15%~~ → 상향 조정된 23% 달성
결과
이번 분기 실적은 전년 대비 23% 성장했으며, 특히 동남아 시장에서 큰 성과를 보였습니다. 기존 목표 15% → 상향 조정된 23% 달성
부분 강조 팁
문장 전체가 아니라 핵심 단어만 감싸세요. "이 부서의 **핵심 과제**는 디지털 전환입니다" — 이렇게 하면 AI도 어디가 중요한지 정확히 인식합니다.

코드 (Code) 필수

코드 문법은 프로그래머만 쓰는 것이 아닙니다. AI와 소통할 때 "이 부분은 하나의 덩어리"라고 알려주는 구분 기호입니다.

인라인 코드 — 백틱(`) 하나로 감싸기

입력 방법렌더링 결과활용 예시
`Ctrl + C` Ctrl + C 단축키 표시
`매출액 = 단가 × 수량` 매출액 = 단가 × 수량 수식·공식 표시
`config.yaml` config.yaml 파일명·함수명 표시

코드 블록 — 백틱 3개(```)로 감싸기

입력
```python
def calculate_growth(current, previous):
    return (current - previous) / previous * 100
```
결과 (구문 강조 적용)
def calculate_growth(current, previous):
    return (current - previous) / previous * 100
입력
```sql
SELECT department, SUM(revenue) as total
FROM sales_2025
GROUP BY department
ORDER BY total DESC;
```
결과 (SQL 구문 강조)
SELECT department, SUM(revenue) as total
FROM sales_2025
GROUP BY department
ORDER BY total DESC;
AI 프롬프트에서 코드 블록의 의미
코드 블록(```)은 AI에게 "이 안의 내용은 하나의 맥락 경계"라고 알려줍니다. 데이터나 예시를 코드 블록으로 감싸면 AI가 지시사항과 데이터를 혼동하지 않습니다. 또한 렌더링된 코드 블록에는 복사(Copy) 버튼이 표시되어 결과를 바로 활용할 수 있습니다.

달러 기호($)로 수식을 표현합니다. LaTeX 문법을 사용합니다.

지원 도구
수학 수식은 Obsidian, Jupyter Notebook, GitHub, Notion 등에서 지원됩니다. 일반 텍스트 에디터에서는 렌더링되지 않습니다. 일상 업무에서 직접 사용할 일은 드물지만, AI가 수식을 포함한 답변을 줄 때 읽을 수 있으면 좋습니다.

왜 마크다운인가?
AI는 #, -, **, ``` 같은 구조 기호를 인식하여 프롬프트의 각 부분을 명확히 구분합니다. 같은 내용이라도 마크다운으로 구조화하면 AI가 역할, 요청, 형식, 제약을 정확히 파악하여 훨씬 체계적인 결과를 만들어냅니다.
